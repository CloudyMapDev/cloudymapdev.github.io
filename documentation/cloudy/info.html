Cloudy

Tip! If you do not have it, you can use /gui in order to get a main menu tool to manage worlds.
Cloudy is a map managing plugin coded by ShinyDialga.
Other plugins like Multiverse were good, but had a lot of flaws.
Cloudy is able to keep track of all maps created by players.
It can even import old maps to a new and better format.
There are several things you can do with Cloudy, take a look at the other pages to learn more.
